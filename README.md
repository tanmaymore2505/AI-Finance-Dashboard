# AI-Finance-Dashboard
An AI-powered personal expense tracking dashboard built with Python and Streamlit.

This project helps users track daily expenses, analyze spending patterns, and predict future expenses using machine learning.

## 🚀 Features

✔ Add and manage daily expenses  
✔ Expense history dashboard  
✔ Category-wise spending analysis  
✔ Monthly spending trend visualization  
✔ AI-based expense prediction using Linear Regression  
✔ Budget monitoring and alerts  
✔ Interactive web dashboard

## 🧠 AI Component

The project uses a **Machine Learning model (Linear Regression)** from scikit-learn to predict potential future expenses based on spending trends.

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn
- NumPy

## 📊 Dashboard Preview

The dashboard includes:

- Expense tracking table
- Category spending charts
- Monthly expense trends
- AI prediction results
- Budget monitoring system
